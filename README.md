# The Best Property Group Website

A modern, beautiful Next.js website for The Best Property Group built with TypeScript, Tailwind CSS, and modern React features.

## 🚀 Features

- **Modern UI/UX**: Clean, professional design inspired by the Axxel landing page
- **TypeScript**: Fully typed for better developer experience and code quality
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development
- **Responsive Design**: Optimized for all devices and screen sizes
- **Animation Libraries**: Includes AOS, React Animate Height, and React Fast Marquee
- **Component-Based**: Organized structure with reusable components

## 🛠️ Tech Stack

- **Framework**: Next.js 16.1.6
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **UI Components**: React 19
- **Animations**: AOS, React Animate Height, React Fast Marquee
- **Styled Components**: For complex styling needs

## 📦 Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

## 🏃‍♂️ Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 🏗️ Building for Production

```bash
npm run build
npm start
```

## 🎨 Design Philosophy

This website is designed to look as good as the Axxel landing page while maintaining its own unique identity for The Best Property Group. It features:

- Modern gradient backgrounds
- Smooth animations and transitions
- Professional color scheme (Blue theme)
- Clear call-to-actions
- Mobile-first responsive design
